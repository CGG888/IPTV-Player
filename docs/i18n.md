# 国际化 (I18n) 说明

**SrcBox (源匣)** 欢迎全球用户参与翻译贡献。本项目目前支持以下语言：

- **English (en-US)** - 默认语言
- **简体中文 (zh-CN)**
- **繁体中文 (zh-TW)**
- **俄语 (ru-RU)**

## 如何添加新语言

项目的多语言支持基于 WPF 的 `ResourceDictionary` 实现。所有翻译文件位于 `Resources/` 目录下。

### 1. 创建翻译文件

1.  复制 `Resources/Strings.en-US.xaml` 文件。
2.  将其重命名为目标语言代码，例如 `Strings.fr-FR.xaml`（法语）。
3.  打开新文件，保留 `x:Key` 不变，将 `<sys:String>` 标签内的文本翻译为目标语言。

```xml
<!-- Resources/Strings.fr-FR.xaml -->
<sys:String x:Key="UI_TabInterface">Interface</sys:String>
<!-- ... -->
```

### 2. 注册新语言

为了让程序识别并加载新的语言文件，需要修改 `App.xaml.cs` 中的 `ApplyLanguage` 方法。

找到以下代码块并添加新的判断逻辑：

```csharp
// App.xaml.cs
public static void ApplyLanguage(string lang)
{
    // ...
    string code;
    if (ci.Name.StartsWith("zh", StringComparison.OrdinalIgnoreCase))
        code = ci.Name.IndexOf("TW", StringComparison.OrdinalIgnoreCase) >= 0 ? "zh-TW" : "zh-CN";
    else if (ci.Name.StartsWith("ru", StringComparison.OrdinalIgnoreCase))
        code = "ru-RU";
    // 添加新的语言判断
    else if (ci.Name.StartsWith("fr", StringComparison.OrdinalIgnoreCase))
        code = "fr-FR";
    else
        code = "en-US"; // 默认回退到英文
    
    // ...
}
```

### 3. 更新设置界面

如果需要在设置界面的语言下拉菜单中显示新语言，请修改 `SettingsWindow.xaml` 或相关绑定源，添加新的选项。

## 提交贡献

完成翻译后，请提交 Pull Request 到 GitHub 仓库。请确保：

1.  文件名符合 `Strings.{code}.xaml` 格式。
2.  所有 `x:Key` 都存在且未被修改。
3.  XML 格式正确无误。

感谢您的贡献！
