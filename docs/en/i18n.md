# Internationalization (I18n)

**SrcBox** welcomes contributions from around the world. The project currently supports the following languages:

- **English (en-US)** - Default
- **Simplified Chinese (zh-CN)**
- **Traditional Chinese (zh-TW)**
- **Russian (ru-RU)**

## How to Add a New Language

Multi-language support is implemented using WPF `ResourceDictionary`. All translation files are located in the `Resources/` directory.

### 1. Create Translation File

1.  Copy `Resources/Strings.en-US.xaml`.
2.  Rename it to your target language code, e.g., `Strings.fr-FR.xaml` (for French).
3.  Open the new file, keep `x:Key` unchanged, and translate the text inside `<sys:String>` tags.

```xml
<!-- Resources/Strings.fr-FR.xaml -->
<sys:String x:Key="UI_TabInterface">Interface</sys:String>
<!-- ... -->
```

### 2. Register New Language

To enable the application to load the new language file, you need to modify the `ApplyLanguage` method in `App.xaml.cs`.

Find the following code block and add logic for the new language:

```csharp
// App.xaml.cs
public static void ApplyLanguage(string lang)
{
    // ...
    string code;
    if (ci.Name.StartsWith("zh", StringComparison.OrdinalIgnoreCase))
        code = ci.Name.IndexOf("TW", StringComparison.OrdinalIgnoreCase) >= 0 ? "zh-TW" : "zh-CN";
    else if (ci.Name.StartsWith("ru", StringComparison.OrdinalIgnoreCase))
        code = "ru-RU";
    // Add check for new language
    else if (ci.Name.StartsWith("fr", StringComparison.OrdinalIgnoreCase))
        code = "fr-FR";
    else
        code = "en-US"; // Fallback to English
    
    // ...
}
```

### 3. Update Settings UI

To make the new language selectable in the Settings dropdown, update `SettingsWindow.xaml` or the relevant binding source to include the new option.

## Contributing

Once your translation is complete, please submit a Pull Request to the GitHub repository. Ensure that:

1.  The filename follows the `Strings.{code}.xaml` format.
2.  All `x:Key` keys are present and unmodified.
3.  The XML format is valid.

Thank you for your contribution!
