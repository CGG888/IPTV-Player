# Getting Started

## Introduction
SrcBox is a modern, high-performance IPTV player built with C# and WPF, powered by the robust `libmpv` playback engine.

## Installation
1. Download the latest release from GitHub.
2. Run `SrcBox.exe`.
3. Load your M3U playlist.

## Features
- **Fast Channel Switching**: Optimized for quick zapping.
- **EPG Support**: XMLTV format support with automatic updates.
- **Timeshift**: Pause, rewind, and replay live TV (if supported by provider).
- **Customizable**: Dark/Light themes, custom logos, and more.
