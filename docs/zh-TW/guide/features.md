# 功能特性

## 核心播放功能
- **播放控制**: 播放/暫停、停止、快進/快退、拖動進度。
- **音量調節**: 滑塊調節，支援靜音（0-100）。
- **狀態標識**: 全螢幕條顯示直播/回放/時移狀態。

## IPTV 專用功能

### M3U 解析
支援本地/遠端 M3U 清單，相容 UTF-8/GB18030 編碼。支援 `#EXTINF` 擴展屬性。

### EPG 節目單
支援 XMLTV (gz) 格式，按日切換，自動匹配 `tvg-id`。
[查看使用與設定說明](./epg)

### 回放（Catchup）
基於模板自動生成回放 URL（如 `{utc:yyyyMMddHHmmss}`）。
[查看使用與自訂說明](./catchup-timeshift)

### 倍速播放
- 僅支援「時移／回放」，直播不支援倍速
- 可選速度：0.5×、0.75×、1.0×、1.25×、1.5×、1.75×、2.0×、3.0×、5.0×
- 選擇後自動開啟音高校正，確保人聲自然不變調
- 全螢幕與視窗模式一致；樣式隨深／淺色主題自適應

### 頻道管理
分組、搜尋、收藏、歷史記錄（本機持久化）。

### 頻道列表標識（R/T）
- R：支援回放；T：支援時移
- 滑鼠懸停具國際化提示（中／英／繁／俄）
- 全螢幕與視窗模式樣式與佈局一致

### 預約通知與預約播放
- 未來節目提供「預約」入口，區分「僅提醒」與「到點自動播放」策略
- 預約清單支援單一視窗管理、核取方塊批次刪除、全選／反選
- 支援系統匣／下拉／右鍵統一入口，視窗層級在播放器之上（視窗／全螢幕一致）
- 到點提醒與成功提示不搶焦點，通知策略與位置一致

### 精簡模式
- 提供精簡播放器視窗形態，保留頂部互動與核心播放控制
- 精簡模式支援獨立縮放命中、邊緣游標提示、視窗拖曳與尺寸同步
- 精簡模式與視窗／全螢幕狀態聯動，標識、EPG、頻道清單狀態同步

### 直播最佳化（FCC）
FCC 快速切台與 UDP 組播最佳化。

## 全螢幕與懸浮層
- **雙擊全螢幕**: 在影片區域雙擊切換全螢幕。
- **懸浮控制條**: 全螢幕下滑鼠移動到底部自動顯示，支援播放控制與 EPG 切換。
- **側邊抽屜**: 全螢幕下滑鼠貼邊顯示頻道列表（右側）或 EPG（左側）。

## v1.1.2 之後新增與優化

- **EPG 狀態晶片**：節目單右側使用晶片式標識（UI 與「預約」統一）
  - 直播=紅色填充、回放=綠色填充（自動適配深/淺色）
  - 點擊「直播」晶片=立即播放當前頻道直播
  - 當前播放項整行高亮；回放播放中左側另有綠色豎線提示（深色更明顯）
- **預約與通知**
  - 預約清單單一視窗管理；首次開啟跟隨播放器置中，使用者拖動後保留位置
  - 支援核取方塊 + 全選/反選，刪除時優先刪除勾選項
  - 系統匣／下拉／右鍵均可進入預約清單，視窗層級在播放器之上（全螢幕／視窗一致）
  - 到點／成功通知不搶焦點：右下角（到點／成功）＋ 置中（清單）定位策略清晰
- **M3U 管理**
  - 新增「管理 M3U 清單」視窗（與預約清單相同樣式），支援核取方塊批次刪除、單筆編輯
  - 系統匣新增入口；下拉／右鍵與系統匣入口行為一致；全螢幕可見且置於播放器上層
- **系統匣**
  - 常駐圖示；雙擊恢復；選單包含 打開／預約／管理 M3U／設定／退出
- **關閉確認**
  - 退出彈窗「×／ESC」僅關閉彈窗；「是」=退出，「否」=最小化到系統匣（全螢幕先退出全螢幕）
  - 文案三段式國際化（已同步中／英／繁／俄）
- **深淺色適配**
  - 預約清單、M3U 管理、預約對話框、退出對話框在視窗建立後即時套用深／淺色標題列
