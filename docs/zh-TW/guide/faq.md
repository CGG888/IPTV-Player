# 常見問題 (FAQ)

## 一般問題
**Q: 如何切換語言？**
A: 進入「設定 > 介面 > 語言」，選擇您偏好的語言。

**Q: 支援遠端 M3U 檔案嗎？**
A: 支援，您可以載入本地或遠端 URL 的播放清單。

**Q: 為什麼有畫面無聲音？**
A: 部分 IPTV 來源音訊探測較慢，已設定 `probesize=32` 加速起播，可能導致短暫無聲。可嘗試切換音軌或重啟播放。

## 故障排查
**Q: 程式啟動即崩潰？**
A: 請確保 `libmpv-2.dll` 已放置在執行目錄 (`bin\Debug\net8.0-windows\`)。

**Q: EPG 顯示「無數據」？**
A: 檢查 XMLTV URL 是否可訪問，是否為 GZIP 格式，以及 `tvg-id` 是否匹配。
