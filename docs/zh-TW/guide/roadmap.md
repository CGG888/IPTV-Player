# 路線圖 (Roadmap)

我們致力於持續提升 IPTV 觀看體驗。

## 未來計畫

- [ ] **智慧推薦**：基於觀看習慣的 AI 節目推薦。
- [ ] **多視角同屏**：支援多源拼屏觀看（畫中畫/四分屏）。
- [ ] **雲端錄製 (PVR)**：支援連接遠端存儲進行節目錄製。
- [ ] **高級視聽體驗**：HDR10+ 動態元資料支援、8K 120fps 解碼最佳化。
- [ ] **互動功能**：語音彈幕（語音轉字幕）、低延遲雲遊戲入口。
- [ ] **版權保護**：區塊鏈版權校驗（防篡改溯源）。

## 已實現功能

- [x] **時移 (Time-Shift)**：基於 `catchup-source` 的回放式時移，支援即時拖動回看。
- [x] **EPG 電子節目單**：支援 XMLTV (gz) 解析與展示。
- [x] **頻道回看**：基於模板自動生成回放 URL。
- [x] **M3U 解析**：支援本地/遠端清單及 `#EXTINF` 擴展屬性。
- [x] **頻道管理**：分組、搜尋、收藏功能。
- [x] **直播最佳化**：FCC 快速切台、UDP 組播最佳化、自動換源。
- [x] **硬體解碼**：預設開啟 `d3d11va`。
- [x] **UI/UX**：全螢幕懸浮控制、側邊抽屜、多語言支援（中/英）。
