# 國際化 (I18n) 說明

**SrcBox (源匣)** 歡迎全球使用者參與翻譯貢獻。本專案目前支援以下語言：

- **英文 (en-US)** - 預設語言
- **簡體中文 (zh-CN)**
- **繁體中文 (zh-TW)**
- **俄語 (ru-RU)**

## 如何新增新語言

專案的多語言支援基於 WPF 的 `ResourceDictionary` 實作。所有翻譯檔案位於 `Resources/` 目錄下。

### 1. 建立翻譯檔案

1.  複製 `Resources/Strings.en-US.xaml` 檔案。
2.  將其重新命名為目標語言代碼，例如 `Strings.fr-FR.xaml`（法語）。
3.  打開新檔案，保留 `x:Key` 不變，將 `<sys:String>` 標籤內的文字翻譯為目標語言。

```xml
<!-- Resources/Strings.fr-FR.xaml -->
<sys:String x:Key="UI_TabInterface">Interface</sys:String>
<!-- ... -->
```

### 2. 註冊新語言

為了讓程式識別並載入新的語言檔案，需要修改 `App.xaml.cs` 中的 `ApplyLanguage` 方法。

找到以下程式碼區塊並新增新的判斷邏輯：

```csharp
// App.xaml.cs
public static void ApplyLanguage(string lang)
{
    // ...
    string code;
    if (ci.Name.StartsWith("zh", StringComparison.OrdinalIgnoreCase))
        code = ci.Name.IndexOf("TW", StringComparison.OrdinalIgnoreCase) >= 0 ? "zh-TW" : "zh-CN";
    else if (ci.Name.StartsWith("ru", StringComparison.OrdinalIgnoreCase))
        code = "ru-RU";
    // 新增新的語言判斷
    else if (ci.Name.StartsWith("fr", StringComparison.OrdinalIgnoreCase))
        code = "fr-FR";
    else
        code = "en-US"; // 預設回退到英文
    
    // ...
}
```

### 3. 更新設定介面

如果需要在設定介面的語言下拉選單中顯示新語言，請修改 `SettingsWindow.xaml` 或相關繫結源，新增新的選項。

## 提交貢獻

完成翻譯後，請提交 Pull Request 到 GitHub 倉庫。請確保：

1.  檔名符合 `Strings.{code}.xaml` 格式。
2.  所有 `x:Key` 都存在且未被修改。
3.  XML 格式正確無誤。

感謝您的貢獻！
