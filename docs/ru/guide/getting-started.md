# Начало работы

## Введение
SrcBox — это современный высокопроизводительный IPTV‑плеер на C# и WPF, построенный на надёжном движке воспроизведения `libmpv`.

## Установка
1. Скачайте последний релиз на GitHub.
2. Запустите `SrcBox.exe`.
3. Загрузите ваш плейлист M3U.

## Возможности
- **Быстрое переключение каналов**: оптимизация для мгновенного zapping.
- **Поддержка EPG**: поддержка формата XMLTV с автоматическим обновлением.
- **Timeshift**: пауза, перемотка и просмотр прямого эфира (если поддерживается провайдером).
- **Гибкая настройка**: светлая/тёмная тема, пользовательские логотипы и другое.
