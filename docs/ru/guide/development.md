# Руководство разработчика

## Требования к среде

- **ОС**: Windows 10 / 11 (x64)
- **IDE**: Visual Studio 2022 или JetBrains Rider
- **SDK**: .NET 8.0 SDK
- **Зависимости**: `libmpv-2.dll` (должен быть вручную помещен в выходную директорию)

## Сборка и запуск

```powershell
# Восстановление зависимостей
dotnet restore

# Сборка (Debug)
dotnet build

# Запуск
dotnet run
```

> **Примечание**: Перед запуском убедитесь, что `libmpv-2.dll` находится в директории `bin\Debug\net8.0-windows\`, иначе программа завершится с ошибкой.

## Устранение неполадок

| Проблема | Возможная причина | Решение |
| :--- | :--- | :--- |
| **Крах при запуске** | Отсутствует `libmpv-2.dll` | Скачайте dll соответствующей архитектуры и поместите в директорию запуска |
| **Видео есть, звука нет** | Таймаут определения аудио | Нормальная стратегия оптимизации, попробуйте переключить дорожку или перезапустить |
| **EPG "Нет данных"** | Проблема с сетью или форматом | Проверьте доступность XMLTV URL и формат GZIP |
| **Настройки не сохраняются** | Нет прав доступа | Убедитесь, что у программы есть права на запись в директорию |
