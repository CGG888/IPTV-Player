# Функциональные возможности

## Основные функции воспроизведения
- **Управление**: Воспроизведение/Пауза, Стоп, Перемотка, Перетаскивание прогресса.
- **Громкость**: Регулировка ползунком, поддержка отключения звука (0-100).
- **Индикация**: Отображение статуса Прямой эфир / Архив / Timeshift.

## Функции IPTV

### Парсинг M3U
Поддержка локальных и удаленных M3U плейлистов, совместимость с кодировками UTF-8/GB18030. Поддержка расширенных атрибутов `#EXTINF`.

### Телегид (EPG)
Поддержка формата XMLTV (gz), переключение по дням, автоматическое сопоставление `tvg-id`.
[Руководство по использованию и настройке](./epg)

### Архив (Catchup)
Автоматическая генерация URL для просмотра архива на основе шаблонов (например, `{utc:yyyyMMddHHmmss}`).
[Руководство по использованию и настройке](./catchup-timeshift)

### Скорость воспроизведения
- Поддерживается только для Timeshift и Архива; для прямого эфира скорость не применяется
- Доступные скорости: 0.5×, 0.75×, 1.0×, 1.25×, 1.5×, 1.75×, 2.0×, 3.0×, 5.0×
- Включается коррекция высоты звука (pitch), чтобы сохранить естественные голоса
- Единый UI в полноэкранном и оконном режимах; адаптация к светлой/тёмной теме

### Управление каналами
Группировка, поиск, избранное (действует во время работы программы).

### Значки R/T списка каналов
- R: доступен архив (Catchup); T: доступен timeshift
- Подсказки по наведению локализованы (ZH/EN/ZH-TW/RU)
- Единый стиль и размещение в полноэкранном и оконном режимах
### Оптимизация (FCC)
Быстрое переключение каналов (FCC) и оптимизация UDP multicast.

## Полноэкранный режим и оверлей
- **Двойной клик**: Переключение в полноэкранный режим двойным кликом по видео.
- **Оверлей управления**: Автоматически появляется внизу экрана при движении мыши в полноэкранном режиме, поддерживает управление воспроизведением и EPG.
- **Боковая панель**: В полноэкранном режиме при наведении мыши на край экрана отображается список каналов (справа) или EPG (слева).

## Обновления начиная с v1.1.2

- **Статусные чипы EPG**: Значки в виде чипов справа от каждой строки программы
  - Прямой эфир = красный заполненный чип; Архив = зеленый заполненный чип (автоподстройка к светлой/темной теме)
  - Клик по чипу “Прямой эфир” мгновенно запускает текущий канал
  - Воспроизводимая программа подсвечивается; при архивном воспроизведении слева отображается зеленая полоса (лучше заметно в темной теме)
- **Напоминания и уведомления**
  - Окно списка напоминаний — единый экземпляр; первое открытие центрируется относительно плеера; пользовательская позиция сохраняется
  - Чекбоксы + “Выбрать все/Инвертировать” для пакетного удаления
  - Доступ через трей/выпадающее/контекстное меню с единым поведением
  - Тосты “успешно/к началу передачи” не перехватывают фокус (нижний правый угол vs. центр)
- **Управление M3U**
  - Новое окно “Управление списком M3U” (в стиле окна напоминаний): пакетное удаление и одиночное редактирование
  - Пункт в системном трее; одинаковые пункты в выпадающем/контекстном меню; окно видно поверх плеера даже в полноэкранном режиме
- **Системный трей**
  - Постоянная иконка; двойной клик — восстановление; меню: Открыть/Напоминания/Управление M3U/Настройки/Выход
- **Подтверждение закрытия**
  - “×/ESC” — только закрытие диалога; “Да” — выход; “Нет” — сворачивание в трей (предварительно выход из полноэкранного режима)
  - Трехстрочная локализация синхронизирована (ZH/EN/ZH-TW/RU)
- **Синхронизация тем**
  - Заголовки окон (Напоминания, Управление M3U, Диалог напоминаний, Диалог выхода) применяют тему на этапе OnSourceInitialized для мгновенной адаптации
