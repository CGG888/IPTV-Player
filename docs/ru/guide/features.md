# Функциональные возможности

## Основные функции воспроизведения
- **Управление**: Воспроизведение/Пауза, Стоп, Перемотка, Перетаскивание прогресса.
- **Громкость**: Регулировка ползунком, поддержка отключения звука (0-100).
- **Индикация**: Отображение статуса Прямой эфир / Архив / Timeshift.

## Функции IPTV

### Парсинг M3U
Поддержка локальных и удаленных M3U плейлистов, совместимость с кодировками UTF-8/GB18030. Поддержка расширенных атрибутов `#EXTINF`.

### Телегид (EPG)
Поддержка формата XMLTV (gz), переключение по дням, автоматическое сопоставление `tvg-id`.

### Архив (Catchup)
Автоматическая генерация URL для просмотра архива на основе шаблонов (например, `{utc:yyyyMMddHHmmss}`).

## Полноэкранный режим и оверлей
- **Двойной клик**: Переключение в полноэкранный режим двойным кликом по видео.
- **Оверлей управления**: Автоматически появляется внизу экрана при движении мыши в полноэкранном режиме, поддерживает управление воспроизведением и EPG.
- **Боковая панель**: В полноэкранном режиме при наведении мыши на край экрана отображается список каналов (справа) или EPG (слева).
