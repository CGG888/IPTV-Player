# Настройка

## `user_settings.json`

Файл находится в директории программы и хранит пользовательские настройки.

```json
{
  "Hwdec": true,              // Аппаратное декодирование
  "SourceTimeoutSec": 3,      // Таймаут переключения источника (сек)
  "TimeshiftHours": 2,        // Длительность Timeshift (часы)
  "Language": "ru-RU",        // Язык интерфейса
  "ThemeMode": "Dark"         // Тема (Dark/Light/System)
}
```

## Описание параметров

- **Hwdec**: Включить аппаратное ускорение (по умолчанию `d3d11va`).
- **SourceTimeoutSec**: Таймаут ожидания источника (в секундах), если источник недоступен, будет попытка переключения на следующий.
- **TimeshiftHours**: Максимальная длительность перемотки назад (в часах).
- **Language**: Код языка интерфейса (например, `ru-RU`, `en-US`).
- **ThemeMode**: Тема приложения (Темная/Светлая/Системная).
