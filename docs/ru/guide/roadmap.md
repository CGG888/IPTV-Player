# Дорожная карта (Roadmap)

Мы стремимся постоянно улучшать опыт просмотра IPTV.

## Планы на будущее

- [ ] **Умные рекомендации**: AI рекомендации программ на основе истории просмотров.
- [ ] **Мульти-просмотр**: Поддержка просмотра нескольких источников одновременно (Картинка-в-картинке / Сетка).
- [ ] **Облачная запись (PVR)**: Поддержка записи передач в удаленное хранилище.
- [ ] **Улучшенное A/V**: Поддержка динамических метаданных HDR10+, оптимизация декодирования 8K 120fps.
- [ ] **Интерактивность**: Голосовые комментарии (Speech-to-Text), вход в облачный гейминг с низкой задержкой.
- [ ] **Защита прав**: Проверка авторских прав на блокчейне.

## Реализованные функции

- [x] **Timeshift**: Сдвиг во времени на основе `catchup-source` с перемоткой в реальном времени.
- [x] **EPG**: Парсинг и отображение XMLTV (gz).
- [x] **Архив (Catchup)**: Автоматическая генерация URL архива по шаблонам.
- [x] **Парсинг M3U**: Поддержка локальных/удаленных плейлистов и атрибутов `#EXTINF`.
- [x] **Управление каналами**: Группировка, поиск, избранное.
- [x] **Оптимизация**: Быстрое переключение FCC, оптимизация UDP multicast, авто-переключение источников.
- [x] **Аппаратное декодирование**: `d3d11va` включено по умолчанию.
- [x] **UI/UX**: Полноэкранный оверлей, боковая панель, многоязычность (CN/EN/RU/TW).
