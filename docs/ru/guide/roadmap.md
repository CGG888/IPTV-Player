# Дорожная карта (Roadmap)

Мы стремимся постоянно улучшать опыт просмотра IPTV.

## Планы на будущее

- [ ] **Умные рекомендации**: AI рекомендации программ на основе истории просмотров.
- [ ] **Мульти-просмотр**: Поддержка просмотра нескольких источников одновременно (Картинка-в-картинке / Сетка).
- [ ] **Облачная запись (PVR)**: Поддержка записи передач в удаленное хранилище.
- [ ] **Улучшенное A/V**: Поддержка динамических метаданных HDR10+, оптимизация декодирования 8K 120fps.
- [ ] **Интерактивность**: Голосовые комментарии (Speech-to-Text), вход в облачный гейминг с низкой задержкой.
- [ ] **Защита прав**: Проверка авторских прав на блокчейне.

## Реализованные функции

- [x] **Timeshift**: Сдвиг во времени на основе `catchup-source` с перемоткой в реальном времени.
- [x] **EPG**: Парсинг и отображение XMLTV (gz).
- [x] **Архив (Catchup)**: Автоматическая генерация URL архива по шаблонам.
- [x] **Парсинг M3U**: Поддержка локальных/удаленных плейлистов и атрибутов `#EXTINF`.
- [x] **Управление каналами**: Группировка, поиск, избранное.
- [x] **Оптимизация**: Быстрое переключение FCC, оптимизация UDP multicast, авто-переключение источников.
- [x] **Аппаратное декодирование**: `d3d11va` включено по умолчанию.
- [x] **UI/UX**: Полноэкранный оверлей, боковая панель, многоязычность (CN/EN/RU/TW).

- [x] **Обновления взаимодействия v1.1.2+**:
  - Чипы статуса EPG: Прямой эфир = красный заполненный; Архив = зелёный заполненный; клик по чипу “Прямой эфир” сразу запускает текущий канал
  - Индикатор текущего воспроизведения: подсветка строки; зелёная левая полоса при архивном воспроизведении (лучше заметно в тёмной теме)
  - Напоминания: единый экземпляр списка; первое открытие центрируется; чекбоксы + “Выбрать все/Инвертировать” для пакетного удаления; доступ из трея/выпадающего/контекстного меню с единым поведением
  - Управление M3U: пакетное удаление и одиночное редактирование; пункт в трее; окно видно поверх плеера в полноэкранном режиме
  - Постоянный системный трей: Открыть/Напоминания/Управление M3U/Настройки/Выход
  - Подтверждение закрытия: ×/ESC — только закрытие диалога; “Нет” — сворачивание в трей (сначала выход из полноэкранного режима)
  - Синхронизация тёмной/светлой темы заголовков: окна Напоминаний, Менеджера M3U, Диалога напоминаний и Диалога выхода применяют тему сразу после создания
  - Скорость воспроизведения: управление скоростью для Timeshift/Архива (включая 3×/5×) с автоматической коррекцией высоты звука; для прямого эфира скорость не поддерживается

## Предпросмотр / Экспериментальные функции

- [~] Анимации напоминаний/уведомлений: стили и анимации подготовлены; включение в следующих версиях
- [~] Кликабельный чип “Архив”: оценивается UX; планируется сосуществование с “клик по названию программы для воспроизведения”
