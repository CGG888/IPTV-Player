# Интернационализация (I18n)

**SrcBox** приветствует вклад переводчиков со всего мира. В настоящее время проект поддерживает следующие языки:

- **Английский (en-US)** — по умолчанию
- **Китайский (упрощённый, zh-CN)**
- **Китайский (традиционный, zh-TW)**
- **Русский (ru-RU)**

## Как добавить новый язык

Многоязычная поддержка реализована на основе `ResourceDictionary` в WPF. Все файлы переводов находятся в директории `Resources/`.

### 1. Создание файла перевода

1.  Скопируйте файл `Resources/Strings.en-US.xaml`.
2.  Переименуйте его, используя код целевого языка, например, `Strings.fr-FR.xaml` (для французского).
3.  Откройте новый файл, оставьте `x:Key` без изменений и переведите текст внутри тегов `<sys:String>` на целевой язык.

```xml
<!-- Resources/Strings.fr-FR.xaml -->
<sys:String x:Key="UI_TabInterface">Interface</sys:String>
<!-- ... -->
```

### 2. Регистрация нового языка

Чтобы приложение могло распознать и загрузить новый файл языка, необходимо изменить метод `ApplyLanguage` в файле `App.xaml.cs`.

Найдите следующий блок кода и добавьте логику для нового языка:

```csharp
// App.xaml.cs
public static void ApplyLanguage(string lang)
{
    // ...
    string code;
    if (ci.Name.StartsWith("zh", StringComparison.OrdinalIgnoreCase))
        code = ci.Name.IndexOf("TW", StringComparison.OrdinalIgnoreCase) >= 0 ? "zh-TW" : "zh-CN";
    else if (ci.Name.StartsWith("ru", StringComparison.OrdinalIgnoreCase))
        code = "ru-RU";
    // Добавьте проверку для нового языка
    else if (ci.Name.StartsWith("fr", StringComparison.OrdinalIgnoreCase))
        code = "fr-FR";
    else
        code = "en-US"; // Возврат к английскому по умолчанию
    
    // ...
}
```

### 3. Обновление интерфейса настроек

Чтобы новый язык можно было выбрать в выпадающем списке настроек, обновите `SettingsWindow.xaml` или соответствующий источник привязки данных, добавив новую опцию.

## Как отправить перевод

После завершения перевода, пожалуйста, отправьте Pull Request в репозиторий GitHub. Убедитесь, что:

1.  Имя файла соответствует формату `Strings.{code}.xaml`.
2.  Все ключи `x:Key` присутствуют и не изменены.
3.  Формат XML валиден.

Спасибо за ваш вклад!
