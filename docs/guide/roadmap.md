# 路线图 (Roadmap)

我们致力于持续提升 IPTV 观看体验。

## 未来计划

- [ ] 智能推荐：基于观看习惯的 AI 节目推荐
- [ ] 多视角同屏：支持多源拼屏观看（画中画/四分屏）
- [ ] 云端录制 (PVR)：支持连接远程存储进行节目录制
- [ ] 高级视听体验：HDR10+ 动态元数据支持、8K 120fps 解码优化
- [ ] 互动功能：语音弹幕（语音转字幕）、低延迟云游戏入口
- [ ] 版权保护：区块链版权校验（防篡改溯源）
- [ ] 通知动画：预约通知淡入/滑入、居中弹窗动效
- [ ] 芯片可定制：直播/回放芯片图标/色彩/圆角的外观模板化

## 已实现功能

- [x] **时移 (Time-Shift)**：基于 `catchup-source` 的回放式时移，支持实时拖动回看。
- [x] **EPG 电子节目单**：支持 XMLTV (gz) 解析与展示。
- [x] **频道回看**：基于模板自动生成回放 URL。
- [x] **M3U 解析**：支持本地/远程列表及 `#EXTINF` 扩展属性。
- [x] **频道管理**：分组、搜索、收藏功能。
- [x] **直播优化**：FCC 快速切台、UDP 组播优化、自动换源。
- [x] **硬件解码**：默认开启 `d3d11va`。
- [x] **UI/UX**：全屏悬浮控制、侧边抽屉、多语言支持（中/英）。
- [x] **v1.1.2+ 交互更新**：
  - EPG 状态芯片：直播=红填充、回放=绿填充；点击直播芯片即播放
  - 当前播放标识：整行高亮；回放播放中左侧绿色竖线
  - 预约通知：列表单实例、首次居中、复选框批删、托盘/右键/下拉入口一致
  - M3U 管理：批量删除、单条编辑、托盘入口、全屏可见
  - 托盘常驻：打开/预约/管理 M3U/设置/退出
  - 退出确认：×/ESC 仅关闭弹窗；“否”最小化托盘（全屏先退出全屏）
  - 深浅色标题栏：预约列表、M3U 管理、预约提醒、退出框在创建后即时匹配主题
  - 倍速控制：支持时移/回放的倍速播放（含 3×/5×），自动音高校正；直播不支持
